О системе

Кратко об учебном пособии

PyBook ( сокращение от Python Book ) - это ЭУП ( Электронное Учебное Пособие ), созданное по заказу учебного учреждения "Хабаровского Промышленно - Экономического техникума" (КГБ ПОУ ХПЭТ) и предназначенное для обучения студентов такому языку программирования как Python 3. Данное пособие рассчитано на тех, кто только недавно узнал об Python и хочет его изучить. PyBook даст первоначальные знания о данном языке программирования и познакомит с его основами.

Философия PyBook

Данное ЭУП предоставляется совершенно безвозмездно ( т.е. даром ), так как PyBook обладает открытым исходным кодом для любого и каждого. Также данное пособие стремится быть максимально полным и ёмким ( т.е. не требовать никаких внешних ресурсов в своей раскрываемой теме ). Ещё PyBook стремиться к независимости и обособленности, т.е. данному учебнику не нужен сервер или интернет для работы, что может быть полезно.

Открытый исходный код

Данное учебное пособие не имеет скрытого кода, а следовательно его исходный код можно легко скачать из репозитория Git. Так же можно легко добавлять, изменять и удалять главы - для этого достаточно скопировать предыдущую страницу главы и вставив её, написать новую главу. Однако, настоятельно рекомендуется соблюдать базовые правила работы с чужим исходным кодом - оставлять ссылку на источник, указывать автора и соблюдать используемый стиль повествования.
